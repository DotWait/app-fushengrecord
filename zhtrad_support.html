常見問題

  • 資料會遺失嗎？

本應用所有資料皆儲存在本機裝置中， 若刪除 App 或更換裝置，資料可能無法恢復， 建議您自行進行裝置備份。

  • 如何開啟 Face ID 隱私鎖？

您可以在應用設定頁面中開啟 Face ID / Touch ID 保護功能。

  • 為什麼無法取得定位資訊？

請前往 iPhone 系統設定 → 隱私權與安全性 → 定位服務， 開啟浮生日記的定位權限。
