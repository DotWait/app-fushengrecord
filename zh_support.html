常见问题

  • 数据会丢失吗？

应用支持本地存储。

  • 如何开启 Face ID 隐私锁？

可在应用设置页面中开启 Face ID / Touch ID 保护。

  • 为什么无法获取定位？

请前往 iPhone 系统设置 → 隐私与安全性 → 定位服务，开启对应权限。
